Find trending words

Instantly find the occurences of up to 5 select words in a specified input string.

Enhance your application's data driven capabilities with this Java API feature by capturing keyword prevalence in textual data such as news articles, e-commerce product descriptions, social media text and more.

Overview: Invoke the API

In this document, you will learn to invoke the API.

  • Parameters
  • Examples

Parameters

Parameters Type Description
haystack String List of terms that are searched and counted in the haystack parameter.
This can be a sentence, or a list of strings separated by these expressions:
  • blank space ( )
  • quotations (")
  • apostrophe (')
  • tab (\t)
  • newline (\n)
  • backspace (\b)
  • page break (\f)
  • carriage return (\r)
needles List of Strings List of terms that are searched and counted in the haystack parameter.

Examples

To analyze the text, initialize an object with the following parameters.

JSONObject humptyDumpty = new JSONObject();
List<String> keywords = Arrays.asList("Humpty", "Dumpty", "king", "men", "egg");

humptyDumpty.put("haystack","Humpty Dumpty sat on a wall\n
Humpty Dumpty had a great fall\n
All the king's horses and all the king's men\n
Couldn't put Humpty together again");

findNeedles(humptyDumpty.haystack, keywords);

Response:

If findNeedles is called successfully with the above parameters, this is the response:

Humpty: 3
Dumpty: 2
king: 2
men: 1
egg: 0

Troubleshooting responses

If your input list for the needles parameter contains more than 5 words, you will get the following message:

Too many words!

Code improvement

Here are some suggestions as to how the code can be improved for at least minor memory and runtime efficiency:

  • Put this String[] words = haystack.split("[ \"\'\t\n\b\f\r]", 0); outside of the loop so it is only initialized once
  • Instead of compareTo, use .equals
  • Instead of initializing an array, use a temporary variable to count each term occurrence and place it into a hashmap or JSON object at the end
